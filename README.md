# Server Página Web

Código de la página web zurullo.com que se hostea desde mi Raspberry Pi 4 model b.
Tiene las siguientes características:
- Monitorea y grafica el estado físico de la Raspberry Pi 4.
- Muestra las carpetas y archivos en línea de mi carpeta FISICA-USC.
- Muestra imagen en directo a través del Livestream de una webcam.
- Método de Login para el Admin, con su propio panel de control de la web.
- Método de Registro y Login para usuarios.
- Chatroom público sin necesidad de login. Vista de usuarios conectados.
- Repositorio de archivos de datos con opción de creación de csv.
- Calculadora de propagación de errores.

A mayores, tiene las instrucciones de como hacer para actualizar la web tras cambios en los archivos locales del servidor al estar conectada por tunnel a cloudflare.
